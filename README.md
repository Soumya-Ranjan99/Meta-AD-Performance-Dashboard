# Meta-AD-Performance-Dashboard
Overview

The Meta Ads Performance Dashboard is a Power BI project designed to provide actionable insights into Facebook and Instagram advertising campaigns. The dashboard helps marketers and business owners track campaign performance, monitor advertising costs, analyze audience engagement, and optimize marketing strategies through interactive visualizations.

Objectives
Monitor advertising performance across multiple campaigns.
Track key marketing KPIs in a single dashboard.
Identify high-performing campaigns and audiences.
Optimize advertising spend based on data-driven insights.
Improve return on investment (ROI) and campaign effectiveness.
